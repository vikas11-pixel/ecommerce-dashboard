# E-Commerce Admin Dashboard

A modern, fully responsive admin panel built with **React.js + Tailwind CSS + Recharts**

### Features
- Sales analytics dashboard with interactive charts
- Product management table (CRUD ready)
- Dark mode toggle
- Responsive sidebar & mobile friendly
- Real-time data visualization
- Clean and professional UI

### Tech Stack
- React 18 + Vite
- Tailwind CSS
- Recharts (Line, Pie, Bar charts)
- Lucide React Icons
